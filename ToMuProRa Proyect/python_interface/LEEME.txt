INICIO RAPIDO
=============

1. Instalar dependencias:
   > pip install -r requirements.txt

2. Iniciar servidor (elegir uno):
   > start_web.bat
   o
   > python web_server.py

3. Abrir navegador:
   http://localhost:8000

CONFIGURACION
=============

Editar config.py para cambiar:
  - Puerto del servidor (default: 8000)
  - Baudrate serial (default: 115200)
  - Parametros por defecto del filtro
  - Limites de rangos


ENDPOINTS API
=============

Serial:
  GET  /api/ports              Lista puertos
  POST /api/connect            Conectar
  POST /api/disconnect         Desconectar

Filtro:
  POST /api/calculate-filter   Calcular coeficientes
  POST /api/send-to-arduino    Enviar al Arduino
  POST /api/reset-arduino      Reset a default
  GET  /api/default-parameters Valores default

WebSocket:
  WS   /ws                     Mensajes en tiempo real
