ESTRUCTURA DEL PROYECTO - INTERFAZ WEB
======================================

│   ├── config.py                  # Configuración centralizada
│   └── __init__.py                # Inicialización del paquete
│
├── Frontend (Web)
│   ├── templates/
│   │   └── index.html            # Interfaz HTML principal

│
├── Configuración
│   ├── requirements.txt           # Dependencias Python
│   ├── .gitignore                # Archivos a ignorar en git
│   └── start_web.bat             # Script de inicio rápido
│
└── Total: 12 archivos funcionales

INICIO RÁPIDO:
1. pip install -r requirements.txt
2. python web_server.py (o start_web.bat)
3. Abrir http://localhost:8000

CONFIGURACIÓN:
- Editar config.py para cambiar puerto, baudrate, etc.
